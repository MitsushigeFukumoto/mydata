行動制御メソッド

  • private int calcAttackPoint(Enemy e)
    • メソッドを呼び出した勇者が、引数の敵に与えられるダメージを計算して返します。（詳細は後述）
  • public int attack(Enemy e)
    • 攻撃を行う。calcAttackPoint()にて計算されたダメージ量を引数の相手に与えます。(引数のオブジェクトのdamage()メソッドを呼ぶ必要があります)この関数の戻り値は与えたダメージ(calcAttackPoint()で計算された値をそのまま返します。)
  • public int damage(int d)
    • ダメージを受ける。対象はメソッドを呼び出した勇者クラスのインスタンス。
      フィールドのhpから引数のダメージ分を減算し、0を下回る場合はhpに0を代入する。
      戻り値はダメージを受けた後のhp。

ダメージ計算詳細

勇者 → 敵

敵キャラクターへ与えられるダメージ計算は以下の計算に従う。

条件（勇者のatk : 敵のdef） ダメージ算出
> 勇者のatk × 1.4(小数点以下切り上げ)
== 勇者のatk
< 勇者のatk × 0.6(小数点以下切り上げ)

上記計算したダメージから、ランダムに-3〜+3の変化を持たせます。

また、ダメージ量が0より下回ることはありません。

敵 → 勇者

勇者キャラクターへ与えられるダメージ計算は以下の計算に従う。

条件（敵のatk : 勇者のdef） ダメージ算出
> 敵のatk × 1.4(小数点以下切り上げ)
== 敵のatk
< 敵のatk × 0.6(小数点以下切り上げ)

上記計算したダメージから、ランダムに-3〜+3の変化を持たせます。

また、ダメージ量が0より下回ることはありません。

小数点以下切り上げ

Math.ceil()メソッドで切り上げができます。

攻撃メソッド(attack)について

勇者、敵クラスそれぞれのattackメソッドでは、与えるダメージを各オブジェクトのcalcAttackPointメソッドの戻り値から取得し、攻撃対象となる相手のオブジェクトのdamageメソッドでダメージを与えてください。

攻撃成功率は、現時点では100%とします。

attack()メソッドの戻り値は、calcAttackPoint()で算出された値を返してください。※実際に与えたダメージ量ではありません。

勇者や敵を生成するクラス(CharacterFactory.java)

勇者クラスのインスタンスや、敵クラスのインスタンスを生成して返すStaticなメソッドを持つクラス

勇者、敵などのキャラクターは、原則このクラスのクラスメソッドを用いて生成します。

フィールド

  • private static final String[] ENEMY_NAMES = {"敵1", "敵2", "敵3", "敵4", "敵5"};
    • 敵キャラクターの名前は、この配列よりランダムで選ばれます。

コンストラクタ

なし

メソッド

  • public static Hero createHero(String name)
    • 勇者を生成して返すメソッド
    • 内部処理は、名前は引数の名前を使用し、それ以外のパラメータは後述するランダムでのパラメータ決定に従いインスタンスを生成し返します
  • public static Enemy createEnemy()
    • 内部処理は、名前はENEMY_NAMESの中からランダムに選んで使用し、それ以外のパラメータは後述するランダムでのパラメータ決定に従いインスタンスを生成し返します
