課題1

クリックカウンターの作成

クリックした回数を表示するプログラムを作成します。

課題内容

プロジェクト名は「ClickCounter」としてください。

  1. フォーム上に9個のラベル を配置します。細かい設定値は各自自由です。
  2. それぞれのラベルに以下のプロパティを設定します。
    • レイアウト→縦方向サイズ、横方向サイズ : 正方形(縦横ともに同じ値)
    • 文字の配置(horizontalAlignment) : 任意
    • 文字の大きさ(font) : 任意
    • 境界線(border) : 何らかの境界線を表示させる（これを設定しないとどこをクリックすればよいか分からなくなります。）
  3. 各ラベルはテキストの初期表示を「0」とし、ラベルがクリックされた時、表示内容がインクリメントされるようにイベントを記述してください。

課題の提出について

  1. 提出するファイルは、実行可能形式のjarファイル とメインメソッドを持つメインクラスのjavaファイル です。
  2. プロジェクトをビルド後、生成されたjarファイルのファイル名を以下の様に変更して提出してください。
    • 「k00000-ClickCounter.jar」:k00000は自分の学籍番号とする。
    • 「k00000-ClickCounter.java」:k00000は自分の学籍番号とする。

課題2

パネル揃えるゲーム

課題1の応用です。 プロジェクトディレクトリごとコピーをするか、別のプロジェクトとして作成後、必要なファイルをコピペ、リネームなどで対応してください。

課題内容

プロジェクト名は、「PanelCompleteGame」とします。

  1. 課題2と同様に、9個のラベルを配置する。
  2. 各ラベルの初期表示は、0か、1かでランダムとする。
    • ランダムな0,1を生成する場合は、int r = String.valueOf(new java.util.Random().nextInt(2)) を使うと良いでしょう。
  3. それぞれのラベルをクリックすると、クリックしたラベルと、上下左右にあるラベルの0,1を反転する。
  4. ラベルをクリック後反転した結果、すべてのラベルが1になった場合に、ゲームクリアとする。
    • ゲームクリアのダイアログメッセージを表示し、再度ランダムで配置する。
  5. リセットボタン等を画面上に配置して、リセットボタン押下した際の処理は初期状態(ランダムに0,1がセットされた状態)になるようにしてください。
    • ランダムな0,1を生成する場合は、int r = String.valueOf(new java.util.Random().nextInt(2)) を使うと良いでしょう。(↑の2番へ移動しました。)

※その他細かい仕様については、講義内で解説します。

補足情報

画面の初期化について

JFrameのイベントに「WindowOpend」があります。 このイベントは、JFrameの画面が開いた時に発火するイベントです。

画面の初期化を行う場合などは、こちらを使うと良いでしょう。 (本来は、別の箇所に記載することもできますが、イベントにて制御する場合はこちらが楽に行えるでしょう。)

ダイアログメッセージを出す場合

ダイアログメッセージの出し方についてですが、javax.swing.JOptionPaneクラスにあるstaticメソッドの、showMessageDialogを使用します。

以下のステートメントでメッセージボックスが出せます。メッセージボックスを出したい箇所に貼り付けてください。

javax.swing.JOptionPane.showMessageDialog(
        this,
        "GameClear!!",                          // メッセージ
        "PanelCompleteGame",                    // タイトル
        javax.swing.JOptionPane.PLAIN_MESSAGE); // ダイアログタイプ
