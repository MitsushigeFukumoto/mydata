成績管理プログラムの作成

管理データクラスの作成

以下のような形式のデータで入力されるプログラムを想定して、入力された(または用意した)データをもとに成績を管理します。

名称 データ管理上の名称 形式
学籍番号 id 文字列(K00000のようなアルファベット1文字+数字5桁のフォーマット)
名前 name 文字列(全角あり)
国語の点数 japanese 整数値(0〜100)
数学の点数 mathematic 整数値(0〜100)
理科の点数 science 整数値(0〜100)
社会科の点数 socialStudy 整数値(0〜100)
英語の点数 english 整数値(0〜100)

上記データは、Scoreクラスで1件分管理します。

Scoreクラスには、以下のメソッドを持たせましょう。

メソッド定義 処理内容
String getId() idのアクセサ
String getName() nameのアクセサ
int getJapanese() japaneseのアクセサ
int getMathematic() mathematicのアクセサ
int getScience() scienceのアクセサ
int getSocialStudy() socialStudyのアクセサ
int getEnglish() englishのアクセサ
int getSum() 5教科の合計点を返す
String getEvaluation() 5教科の合計点に対する評価を文字列で返す(〜450:A, 449〜400:B, 399〜300:C, 299〜200:D, 199〜:E)

これらのメソッドは、テストプログラムで使用しますので実装するべきインターフェイスをこちらで指定します。

以下のインターフェイスをプロジェクトに追加し、Scoreクラスにて実装してください。

インターフェイス名: IScore
パッケージ: Scoreクラスと同じ

/**
 * 第10回課題の点数データ1件分
 */
public interface IScore {
  String getId();
  String getName();
  int getJapanese();
  int getMathematic();
  int getScience();
  int getSocialStudy();
  int getEnglish();

  /**
   * 5教科の合計点を返す
   *
   * @return 5教科の合計点
   */
  int getSum();

  /**
   * 5教科の合計点に対する評価を文字列で返す (〜450:A, 449〜400:B, 399〜300:C, 299〜200:D, 199〜:E)
   *
   * @return A,B,C,D,Eのどれか
   */
  String getEvaluation();
}

また、コンストラクタはinterfaceで指定できませんので、以下のものを各自実装してください。

  • public Score(String id, String name, int japanese, int mathematic, int science, int socialStudy, int english)
    • 引数のそれぞれのデータでフィールドに代入

データソートを行うクラスの作成

成績データの管理と、データソートを行うクラスをScoresSorterとします。

フィールド

  • private List<IScore> data;

必要に応じてprivateで追加すること。

メソッド

  • public List<IScore> getData()
    • フィールドのdataを返す
    • 他のメソッドを呼び出した場合に、このメソッドの戻り値が変わってしまわないように注意すること。
  • public List<IScore> sortBySumDsc()
    • フィールドのdataのコピーを合計点の降順で並び替えを行う。ただし、同じ点数のデータがある場合は、学籍番号のUnicode昇順で並び替えること。
  • public List<IScore> sortByEvaluationAsc()
    • フィールドのdataのコピーを評価のUnicodeの昇順で並び替えを行う。ただし、同じ評価のデータがある場合は、学籍番号のUnicode昇順で並び替えること。

その他必要なメソッドはprivateで追加すること。

コンストラクタ

  • public ScoresSorter(List<IScore> data)
    • 引数のデータでフィールドのdataを初期化すること

コンストラクタは、これ以外作成しないこと。

実装用インターフェイス

import java.util.List;
/**
 * 第10回課題のソートプログラムインターフェイス
 */
public interface IScoresSorter {
  /**
   * フィールドのdataを返す。
   * 他のメソッドを呼び出した場合に、このメソッドの戻り値が変わってしまわないように注意すること。
   * @return
   */
  public List<IScore> getData();
  /**
   * フィールドのdataのコピーを合計点の降順で並び替えを行う。ただし、同じ点数のデータがある場合は、学籍番号のUnicode昇順で並び替えること。
   * @return
   */
  public List<IScore> sortBySumDsc();
  /**
   * フィールドのdataのコピーを評価のUnicodeの昇順で並び替えを行う。ただし、同じ評価のデータがある場合は、学籍番号のUnicode昇順で並び替えること。
   * @return
   */
  public List<IScore> sortByEvaluationAsc();
}
