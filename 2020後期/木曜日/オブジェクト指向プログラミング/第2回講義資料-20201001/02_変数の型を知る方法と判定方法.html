変数（オブジェクト）の型を知る方法と判定方法

type関数

第１週で説明したtype関数を利用して変数の型を知ることができる．

i = 0
f = 0.0
s = ''
a = []
t = ()
h = {}
b = False

print(type(i))
print(type(i) is int)
print(type(f))
print(type(f) is float)
print(type(s))
print(type(s) is str)
print(type(a))
print(type(a) is list)
print(type(t))
print(type(t) is tuple)
print(type(h))
print(type(h) is dict)
print(type(b))
print(type(b) is bool)

isinstance関数

isinstance関数を利用して判定することもできる．

isinstance関数は，この後出てくる継承クラスでもTrueが帰ってくるがtype関数ではFalseとなる．（詳しくは後術）

i = int()
f = float()
s = str()
a = list()
t = tuple()
h = dict()
b = bool()

print(i)
print(isinstance(i, int))
print(f)
print(isinstance(f, float))
print(s)
print(isinstance(s, str))
print(a)
print(isinstance(a, list))
print(t)
print(isinstance(t, tuple))
print(h)
print(isinstance(h, dict))
print(b)
print(isinstance(b, bool))

まとめ

pythonで使えるJavaのプリミティブ型に相当するものを紹介した．

pythonプログラムで変数に格納されているのはこれらの型を内包したリスト型であったり辞書型であることが多い．

プログラム開発においてデバッグ作業は必須でありデバッグ作業中には変数の中身を出力することがあるが，変数の出力書式は上書きでるので型を調べるとい発想を常に頭にいれておいて欲しい．
