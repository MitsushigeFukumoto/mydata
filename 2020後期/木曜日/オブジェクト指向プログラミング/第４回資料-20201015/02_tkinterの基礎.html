tkinterの基礎

ウィンドウにWigetを配置していきます

  • Frame：他のWigetを格納するWiget

  • Label：文字を表示するWiget

  • Entry：入力可能なテキストボックスを表示するWiget

  • Button：クリック可能なボタンを表示するWiget

root - Frame - Label
             ┗ Entry
             ┗ Button

Hello World的なGPUアプリ

hello_tkinter.python

from tkinter import *
from tkinter import ttk

root = Tk()
root.title('Hello World')

# ウィジェットの作成
# まずはフレームを配置する(paddingは引数１つの場合上下左右の余白)
frame1 = ttk.Frame(root, padding=16)
label1 = ttk.Label(frame1, text='名前を入力')
t = StringVar()
entry1 = ttk.Entry(frame1, textvariable=t)
button1 = ttk.Button(
    frame1,
    text='Submit',
    command=lambda: print('Hello, %s.' % t.get()))

# レイアウト
# side=LEFTを指定すると子Wigetを左から順番に並べる
# side=LEFTのところをTOP, RIGHT, BOTTOMに変更してみよう
frame1.pack()
label1.pack(side=LEFT)
entry1.pack(side=LEFT)
button1.pack(side=LEFT)

# ウィンドウの表示開始
root.mainloop()

簡単な解説

packer

  • 参考URL：https://docs.python.org/ja/3.8/library/tkinter.html#packer-options

  • sideの他にも，ancer, expand, fill, ipadx, ipady, padx, padyから指定できる

style config（属性の変更，例：背景色の変更など）

  • 参考URL:https://docs.python.org/ja/3.8/library/tkinter.ttk.html#tkinter.ttk.Style
# 全てのLabelに関するstyleを定義
style = ttk.Style()
style.configure("TLabel", foreground="red")
# button1にのみstyle設定
style = ttk.Style()
style.configure("L1.TLabel", foreground="green")
label1 = ttk.Label(frame1, text='名前を入力', style="L1.TLabel")
# button1のgoregroundのstyleを取り出し変更
style = ttk.Style()
style.configure("BW.TLabel", foreground="blue", background="red")
button1 = ttk.Button(
    frame1,
    text='Submit',
    style="BW.TLabel",
    command=lambda: style.configure("BW.TLabel))
