自作モジュールのインポート

自作のHeroクラスをモジュールとして読み込み(import)して別のプログラムで利用します．

lecture03_hero.pyをmy_moduleというディレクトリにコピーしてください．

  • ディレクトリ構造は以下のようになります
lecture03_magician.py
my_module
    ┗ lectuer03_hero.py

Heroクラスを継承したMagicianクラスを作ります．

# fromでディレクトリとファイル名を指定
# importでクラス名を指定
from my_module.lecture03_hero import Hero

# Heroクラスを継承したMagicianクラスを宣言
class Magician(Hero): 

    # コンストラクタ
    # デフォルト値を指定することもできる
    # なお，引数が異なるコンストラクタを複数定義することはできない
    def __init__(self, name=None, hp=0, mp=0):
        super().__init__(name, hp)
        self.mp = mp

    # 特殊な関数の宣言
    def __str__(self):
        return f'"名前": "{self.name}", "HP": {self.hp + self.job_status["hp"]}, "MP": {self.mp}'


if __name__ == '__main__':
    hero1 = Hero('愛知太郎') # Heroクラスのインスタンスを生成
    print(f'hero1={str(hero1)}') # 文字列に変換
    hero2 = Magician(name='愛知花子', hp=100, mp=100) # 引数名を指定してコンストラクタを呼び出し
    print(f'hero2={str(hero2)}') # 文字列に変換
    print(type(hero1)) # <class 'my_module.lecture03_hero.Hero'>
    print(type(hero2)) # <class '__main__.Magician'>
    if type(hero1) is Hero:
        print('hero1はHero')
    if type(hero2) is not Hero:
        print('hero2とはHeroではない')
    if isinstance(hero2, Hero):
        print('hero2はHeroクラスを継承している')
    if isinstance(hero2, Hero):
        print('hero2はHeroクラスを継承している')

まとめ

自作クラスをモジュールとして読み込む方法を紹介した．

Pythonモジュールのimportは次の順番で読み込むことが推奨されている．

  1. Pythonネイティブもジュール(csv, xml.etree.ElementTreeなど)
  2. pipなどでインストールした外部モジュール(numpyなど)
  3. 自作もジュールの順で読み込むことが推奨されている．
