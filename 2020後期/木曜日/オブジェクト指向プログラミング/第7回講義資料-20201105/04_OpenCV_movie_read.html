OpenCV 動画ファイルの読み込み

OpenCVによるカメラ画像の取得

  • OpenCV公式ドキュメント：http://opencv.jp/opencv-2svn/py/
import cv2

cap = cv2.VideoCapture('uchitane_far.mp4')
print(type(cap))
# <class 'cv2.VideoCapture'>

print(cap.isOpened())
# True

print(cap.get(cv2.CAP_PROP_POS_FRAMES))
# 0.0

print(cap.get(cv2.CAP_PROP_POS_MSEC))
# 0.0

ret, frame = cap.read()

print(ret)
# True

print(type(frame))
# <class 'numpy.ndarray'>

print(frame.shape)
# (720, 1280, 3)

print(cap.get(cv2.CAP_PROP_POS_FRAMES))
# 1.0

print(cap.get(cv2.CAP_PROP_POS_MSEC))
# 0

print(1 / cap.get(cv2.CAP_PROP_FPS) * 1000)
# 40

# 100フレーム目へ
cap.set(cv2.CAP_PROP_POS_FRAMES, 10)
print(cap.get(cv2.CAP_PROP_POS_FRAMES))
# 100.0

まとめ

OpenCVで動画を読み込むサンプルを示した．

OpenCVでは動画の１フレームを画像として扱える．（ただし，音声は処理できないので注意．）
