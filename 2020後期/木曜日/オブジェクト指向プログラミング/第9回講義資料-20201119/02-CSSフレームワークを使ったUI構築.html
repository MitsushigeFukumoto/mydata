CSSフレームワークを使ったUI構築

オブジェクト指向プログラミング及び演習２

Bootstrapを使ったUI構築でアプリケーション開発に集中しよう

CSSで1からアプリケーションのUIを作り上げるのは、かなりの知識と労力を必要とします。 かなりの知識と労力を必要とするにも関わらず、実際のWebシステム開発においては、デザインなどの見た目の構築は後回しにしたほうが機能面の充実を図れるメリットがあります。

そこで登場するのが、CSSフレームワークです。

CSSフレームワークは、JavaやPythonなどのライブラリと同様に人が作ったデザイン済みのCSSを自分のWebアプリケーションに適用することが可能となります。

CSSフレームワークを用いると、アプリケーションのUI構築にて時間が掛かりがちになってしまうデザインの時間をすっ飛ばしてアプリケーションの機能に集中して開発を行うことが可能となります。

ただし、デメリットもあります。主なデメリットは、CSSフレームワーク自体への理解が必要となることです。 このデメリットは、フレームワークを使う上ではどんなものでもついてくるデメリットのため、必ず使用するフレームワークの公式ドキュメントはいつでも閲覧できるようにしておきましょう。

代表的なCSSフレームワーク

  • Bootstrap · The most popular HTML, CSS, and JS library in the world.
    • もともとTwitter社が作ったOSSのCSSフレームワーク
    • Webアプリケーションへの採用率が非常に高く、インターネットに知見も多いため人気がある
  • Bulma: Free, open source, and modern CSS framework based on Flexbox
    • 最近人気のやつ
  • Tailwind CSS - A Utility-First CSS Framework for Rapidly Building Custom Designs
    • カスタマイズしやすい
    • フロントエンドのJavaScriptライブラリ(ReactやVue)と親和性が高い
  • Semantic UI
    • 結構古くからある
  • Pure
    • めっちゃ軽い

他にも様々なCSSフレームワークがあります。どれを使っても良いのですが、複数人での開発を考えた場合には、知名度の高いものだったり、公式ドキュメントが読みやすいものだったりを選ぶと良いでしょう。

Starterテンプレート

以下のテンプレートを適当なhtmlファイルとして保存し、ブラウザで開いてみましょう。※要インターネット接続

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

    <!-- Bootstrap CSS -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@4.5.3/dist/css/bootstrap.min.css" integrity="sha384-TX8t27EcRE3e/ihU7zmQxVncDAy5uIKz4rEkgIXeMed4M0jlfIDPvg6uqKI2xXr2" crossorigin="anonymous">
    <!-- Bootstrap JS(compiled bundle) -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@4.5.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-ho+j7jyWK8fNQe+A12Hb8AhRq26LrZ/JpcUGGOn+Y7RsweNrtN/tE3MoK7ZeZDyx" crossorigin="anonymous"></script>

    <title>Bootstrap Starter template</title>
</head>
<body>
    <h1>Hello, world!</h1>
</body>
</html>

このテンプレートは、一応公式ドキュメントそのままではなく、JavaScriptファイルの配置やhtmlタグのlang属性を変更しています。

このテンプレートを使用して、いくつかコンテンツを追加してみましょう。 ※講義内で解説しながら行います。

(余談)Bootstrapの様々なテーマ

Bootstrapは、Bootstrapを基本としたテーマファイルの存在も大きいです。

  • Bootstrap Themes Built & Curated by the Bootstrap Team.

有料のものが多くありますが、これらのテーマを使うことで更に目的のUI構築を行うことも可能です。

また、Bootstrapは英語圏で作られたものなので、日本語でのコンテンツを配置するには若干の調整が必要だったりします。 オープンソースのプロジェクトに、Bootstrapの日本語カスタマイズ済みのテーマもあります。紹介のみにとどめておきますが、こういったテーマを使うのも良いでしょう。
