Pythonのインストール

まえおき

エラーが起こった場合

  • 画面を見ていてエラーが起こった場合はTAを呼ぶこと

インストール前の確認

  • Python3.8が既にインストール済みな人はインストール不要です
  • pyenvがインストールされている人はpyenvのインストールは不要です
  • 確認方法:ターミナルでコマンドを入力してpythonは3.8が，pyenvはバージョンが表示されればOkです
python3 --version # 3.8であればOK
pyenv --version # バージョンが表示されればOK

Python3.8のインストール

  1. Pythonのバージョン管理ツールpyenvをhomebrewでインストール
brew install pyenv
brew upgrade pyenv
cat << 'EOS' >> ~/.zshrc
export PYENV_ROOT=/usr/local/var/pyenv
if command -v pyenv 1>/dev/null 2>&1; then
eval "$(pyenv init -)"
fi
EOS
# shellをリロードする
zsh -l

  1. pyenv --versionとターミナルに入力してバージョンが表示されることを確認

  2. Python3.8.5をインストール

pyenv install 3.8.5
pyenv global 3.8.5
python3 -m pip install numpy

本日利用するライブラリをインストール

  • numpyをインストール（一度インストールすれば次回からインストールは不要です）
python3.8 -m pip install numpy

(おまけ)pyenvがgit cloneでインストールされている人の更新方法

  • which pyenvと入力すると/Users/xxxx/.pyenvと表示される人向けの話です．
  • pyenv install --listに3.8.5が含まれていない人向けの話です．
cd ~/.pyenv
git pull
cd ~/pyenv/plugins/pyton-build
git pull
zsh -l
pyenv --version
pyenv install 3.8.5
pyenv global 3.8.5
python3.8 -m pip install numpy
