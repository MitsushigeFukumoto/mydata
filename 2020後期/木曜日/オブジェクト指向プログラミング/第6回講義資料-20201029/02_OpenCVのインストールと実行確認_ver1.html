OpenCVモジュールのインストール

  • OpenCVはPythonだけでなくC++など様々なプログラム言語から利用されることが想定されたコンピュータビジョンのライブラリです

  • Pythonから利用するためにはbrewでOpenCVライブラリをシステムにインストールした後，pipでpythonのopencvモジュール(システムのライブラリを読み込む役割)をユーザー領域にインストールする

※ OSのプログラムやライブラリは，PCの全ユーザーがつかえるシステム領域にインストールする方法と，ログインしているユーザーのみが使えるユーザー領域にインストールすることができる

※ システム領域のライブラリを参照するかユーザー領域のライブラリを参照するかは環境変数でユーザーごとに変更する(プログラムが動かないのは結局ユーザーの責任です．嫌ならシステム設定込みでサービスや製品を売っている会社にお金払ってね！)

インストール手順

  1. brewでOpenCVライブラリをシステムにインストールand/orアップグレード
brew install opencv
  • もし，opencvがインストーされているならつぎのようにupgradeを促すメッセージが出力される．その場合はupgradeも実施する．
brew install opencv
・・・・・
######################################################################## 100.0%
Error: opencv 4.5.0 is already installed
To upgrade to 4.5.0_2, run `brew upgrade opencv`.
# 上のようにアップグレードを指示された場合次のコマンドを実行する
brew upgrade opencv
  1. pipでopencvモジュールをインストール（手順1でエラーが出てないことを必ず確認してから実行すること）
python3 -m pip install opencv-python
  1. pyenvで3.8.5を再インストールしopencvモジュールをインストール(１行づつコピーして実行すること　)
pyenv uninstall 3.8.5
pyenv install 3.8.5
pyenv global 3.8.5
python3 --vresion # Python 3.8.5と出力されることを確認
python3 -m pip install numpy opencv-python
  1. opencvをインポートしてエラーが出ないことを確認
# python3 # インタープリターを起動
import cv2
# モジュールがないというエラーがでないことを確認
quit()
